Ссылка на бот: @HelperNerumabot

Интерфейс, если начинать с команды /start, 
и выбирать  из предложенных вариантов - интуитивно понятен.
После того, как будет выбран предмет и уровень сложности:
- их можно изменить в любой момент, используя соответсвующие команды
- любой написанный текст будет отправляться в нейросеть
- после ответа нейросети есть возможность запросить продолжение ответа

Скрытые команды:
debug - Отправляет файл с логами
deleteall - Удаляет все данные из таблицы
